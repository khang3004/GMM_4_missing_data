# Boeing 707 Aviation Accidents Dataset (Since 1948)

## Overview

This dataset showcases Boeing 707 accidents that have occurred since 1948. It provides a comprehensive historical record of aviation safety incidents involving this iconic aircraft type, including detailed information on the circumstances, casualties, and technical details of each event.

## Dataset Utility

This dataset includes information on over **5,000 airplane crashes** around the world. It is an absolutely essential resource for anyone interested in aviation safety. Researchers and enthusiasts can find data on when and where each crash occurred, what type of plane was involved, the human toll, and much more.

## Column Descriptions

| Column           | Description                                                      |
| :--------------- | :--------------------------------------------------------------- |
| **index**        | Unique identifier for each row.                                  |
| **Date**         | The date of the incident.                                        |
| **Time**         | The time of the incident (local or UTC).                         |
| **Location**     | The geographic location of the accident.                         |
| **Operator**     | The airline or organization operating the aircraft.              |
| **Flight #**     | The specific flight number assigned to the trip.                 |
| **Route**        | The planned route of the aircraft (Origin - Destination).        |
| **Type**         | The specific type/model of the aircraft (e.g., Boeing 707-321B). |
| **Registration** | The aircraft's registration mark (Tail number).                  |
| **cn/In**        | Construction number and/or Line number.                          |
| **Aboard**       | Total number of persons on board (passengers and crew).          |
| **Fatalities**   | Number of fatalities among those on board.                       |
| **Ground**       | Number of people killed on the ground.                           |
| **Summary**      | A brief narrative/description of the accident and its causes.    |
